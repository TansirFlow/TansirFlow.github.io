docker部署rustdesk私有服务器

使用docker拉取服务端镜像并运行

docker run --restart always \
--privileged \
-p 9000:9000 \
-p 21114:21114 \
-p 21115:21115 \
-p 21116:21116 \
-p 21116:21116/udp \
-p 21117:21117 \
-p 21118:21118 \
-p 21119:21119 \
-e KEY=你的key \
-e ENABLE_WEB_CLIENT=true --name rustdeskserver -d \
chengxudong2020/rustdeskserver:latest

其中KEY修改为自己想要的值，并且记住这个值

使用桌面客户端远程控制

  1. 安装rustdesk客户端 rustdesk-release

  2. 设置中继服务器

然后输入目标计算机的ID以及密码就可以实现远程控制

使用网页客户端远程控制

访问链接http://服务器ip:9000 进入网页客户端

此处ID服务器和KEY像前面的桌面客户端一样的填写内容

然后输入目标计算机的ID以及密码就可以实现远程控制
