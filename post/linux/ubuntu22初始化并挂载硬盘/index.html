Ubuntu22初始化并挂载硬盘

  1. 查看磁盘信息

    sudo fdisk -l

  1. 找到要挂载的硬盘名称，此处是 /dev/vdb ，然后使用一下命令进行分区
sudo parted /dev/vdb

接着输入p可以查看分区结果，按q可以退出

  1. 再次查看磁盘信息sudo fdisk -l，默认生成dev/vdb1

  1. 挂载硬盘

创建一个目录用于挂在硬盘sudo mkdir /mnt/disk2 （一般挂载在mnt目录下，也可以是别的地方，反正此处挂在目录和后面的要对应上）

赋予权限sudo chmod -R 777 /mnt/disk2

临时手动挂载sudo mount /dev/vdb1 /mnt/disk2 (此处/dev/vdb1改成你自己的分区名称)

如果想开机后磁盘自动挂载，按如下操作

修改文件sudo vim /etc/fstab

添加行/dev/vdb1 /mnt/disk2 ext4 defaults 0 0 (此处/dev/vdb1参考前面的手动挂载方式)
