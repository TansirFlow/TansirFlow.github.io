Ubuntu22设置交换空间文件

增加交换空间

  1. 检查当前的交换空间情况
sudo swapon --show
  1. 创建一个交换空间的文件
sudo fallocate -l 2G /swapfile 
# 注：/swapfile指的是交换文件的路径文件名，可以自己定义，但要与后面的保持一致
# 2G指的是空间大小，自定义
  1. 设置文件权限
sudo chmod 600 /swapfile # 此处文件名与前面的保持一致
  1. 格式化交换文件
sudo mkswap /swapfile # 此处文件名与前面的保持一致
  1. 启用交换空间
sudo swapon /swapfile # 此处文件名与前面的保持一致
  1. 更新 /etc/fstab 文件，使用vim编辑此文件，在末尾增加一行，此处/swapfilea依然要与前面保持一致
/swapfile none swap sw 0 0

删除交换空间

  1. 更新 /etc/fstab 文件，使用vim编辑此文件，删掉你自己增加的那一行
  2. 关闭交换空间
sudo swapoff /swapfile # 此处文件名与前面的保持一致
  1. 删除交换文件
sudo rm /swapfile  # 此处文件名与前面的保持一致
