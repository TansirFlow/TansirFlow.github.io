如何获取Let‘s Encrypt SSL泛型域名证书

以Ubuntu为例

首先安装certbot

sudo apt install certbot

使用certbot获取证书

sudo certbot certonly -d *.example.com –manual –preferred-challenges dns

注：将*.example.com 替换为你的域名

根据提示输入邮箱并回车

输入Y回车

输入N回车

证明域名是你的

按照提示修改DNS解析，解析类型为TXT，值为给出的那一串码

回车确认，域名一般存在于/etc/letsencrypt/live/你的域名/目录下，其中privkey.pem是私钥文件，fullchain.pem是证书文件
